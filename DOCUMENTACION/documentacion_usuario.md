
# Documentación para el Usuario

## Introducción
Esta aplicación permite administrar garages y sus respectivos slots disponibles para estacionamiento. El usuario puede consultar información, ver disponibilidad y realizar modificaciones según los permisos correspondientes.

## Funcionalidades Disponibles

### 1. Ver estado del garage
Muestra todos los pisos y los slots, indicando si están ocupados o disponibles, junto con su tipo (moto, auto, camioneta).

### 2. Actualizar tipo de un slot
Permite modificar el tipo de un slot vacío.  
Opciones del menú:
- Ingresar ID de slot para actualizarlo.
- Presionar **V** para ver los slots disponibles.
- Presionar **S** para salir sin realizar cambios.

El sistema valida:
- Que el ID sea numérico.
- Que el slot no esté ocupado.
- Que el tipo de slot ingresado sea válido.

### 3. Consultar slots disponibles
Permite visualizar rápidamente todos los slots libres dentro del garage.

### 4. Seguridad y validaciones
La aplicación evita:
- Modificaciones sobre slots ocupados.
- Ingresos inválidos.
- Datos incompletos o inconsistentes.

## Beneficios
- Facilita el control del garage.
- Permite mantener la información organizada.
- Evita errores manuales.
- Acelera la gestión de espacios disponibles.
