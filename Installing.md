
# Documentación Técnica para Programadores

## Repositorio
Para más información y actualizaciones, visita el repositorio de GitHub:
`https://github.com/iairkap/programacion1`

## Clonar el proyecto
```bash
git clone [Repo](https://github.com/iairkap/programacion1)
```

## Librerías necesarias
Instalar dependencias (asegurat de tener actualizado pip):
```bash
pip install colorama
```

## Login externo
El proyecto utiliza un módulo externo para el manejo de usuarios disponible en:
`[MODULO FORMIDABLE](https://github.com/pilipayo/Modulo_Formidable.git)`
